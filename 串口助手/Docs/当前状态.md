2. **.csproj 已改为 v4.8**：如果未来有 VS 升级弹窗，不必重复操作
3. **按钮颜色逻辑**：`OpenSerialPort()` 仍会设 `btnOpen.BackColor = Color.Pink`（功能逻辑，未动），`CloseSerialPort()` 恢复为 `Color.White`（已改）
