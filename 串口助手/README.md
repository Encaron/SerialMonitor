# 串口助手（WinForms 原版）

这是 B 站 UP 主**江科大**的 WinForms 串口助手 V1.1。

Encaron 在此代码基础上做了最小改动——把默认波特率从 9600 改为 115200、编码改为 UTF-8、收发模式改为文本。改完之后双击 exe 不用每次手动调，从此开启了"工具应该记住我怎么用"的执念。

**此目录是历史存档，只读不动。** 后续演进见：

```
本版（WinForms 原版） → [串口助手-WPF-v1](../串口助手-WPF-v1/)（第一次 WPF 重写）
                     → [Serial Monitor V2](../Serial%20Monitor%20V2/)（当前活跃版本）
```
