## 架构

```
串口字节 → DataReceived → 文本行 →
  ├── ProtocolParser
  │   ├── [plot,...]    → PlotViewModel      （波形图）
  │   ├── [slider,...]  → SliderPanelViewModel（滑杆）
  │   ├── [key,...]     → KeyPanelViewModel   （按键）
  │   ├── [joystick,...]→ JoystickPanelViewModel（摇杆）
  │   ├── [display,...] → DisplayPanelViewModel（虚拟 OLED）
  │   └── 普通文本      → 接收区（原样显示）
  └── 接收区：AvalonEdit 虚拟化渲染（行为与 v1 完全一致）
```

## 协议格式

```
[类型, 参数1, 参数2, ...]

plot           [plot,曲线名,数值]
plot-clear     [plot-clear]
slider         [slider,名称,百分比]
key            [key,名字,状态]         状态: down / up
joystick       [joystick,编号,h1,v1,h2,v2]
display        [display,x,y,文本,字号,#颜色]
display-clear  [display-clear]
```

## ⚠️ 硬性约束（27 条，改动前必读）

| # | 规则 |
|:--:|------|
| **1** | **4 个数据转换方法禁止修改逻辑**——`BytesToHex`/`HexToBytes`/`BytesToText`/`TextToBytes` 算法一行不准改 |
| **2** | **串口底层操作保留原行为**——`serialPort.Read()`/`Write()`/`Open()`/`Close()` 调用方式不变 |
| **3** | **接收区显示行为完全不变**——`LogReceived`/`LogSent`/`LogSystem`/`AppendColoredLine` 原样 |
| **4** | **用户是新手小白**——编译成 .exe，用户只双击运行 |
| **5** | **Git 已上传云端**——不 force push，中文 commit message |
| **6** | **现有 XAML 设计系统是核心资产**——21 色 DynamicResource、9 套 ControlTemplate、圆角卡片 |
| **7** | **`{DynamicResource}` 必须用 Dynamic**——全 XAML 96 处，新增控件必须 Dynamic |
| **8** | **AvalonEdit 已 NuGet 管理**——编译自动包含，不需要手动复制 DLL |
| **9** | **工作区隔离**——`Serial Monitor V2/` 唯一工作区；`../串口助手-WPF/` 只读 |
| **10** | **编译即交付**——`dotnet publish -c Release` → 单文件 exe 放项目根 |
| **11** | **新 .cs 文件自动包含**——SDK-style csproj 自动扫目录，但 Resource（PNG）需声明 |
| **12** | **v1 功能一个不能丢**——行号/系统日志/智能滚屏/搜索/HEX/定时发送/快捷发送/热插拔/流量统计/崩溃日志 |
| **13** | **Exception 保留原中文错误信息**——用户凭这些消息自己排查 |
| **14** | **🚫 Path.Stroke 禁止 {Binding}**——BAML 反序列化崩溃。用 `Rectangle+OpacityMask+ImageBrush` |
| **15** | **RadioButton 跨容器用 GroupName**——DockPanel+StackPanel 嵌套打破默认互斥 |
| **16** | **csproj 已是 SDK-style**——Phase 6 升级 .NET 8 时转的，不要转回旧格式 |
| **17** | **编译用 dotnet publish**——`PublishSingleFile=true, SelfContained=false`，global.json 锁定 8.0.422 |
| **18** | **侧面板折叠/拖拽共存**——MinWidth=120，折叠时临时降为 0 |
| **19** | **🚫 第三方 WPF 控件不放 XAML**——OxyPlot 等必须在 C# 中 `new`，XAML 只声明命名空间 |
| **20** | **初始选中状态不放 XAML**——`IsChecked`/`SelectedIndex` 在构造器 `InitializeComponent()` 之后设置 |
| **21** | **PlotViewModel 在 LoadWindowSettings 之前初始化**——否则暗色模式颜色失效 |
| **22** | **🚫 未使用的 3rd-party xmlns 禁止在 XAML**——声明了就崩，只声明实际使用的 |
| **23** | **🚫 整理 Git 历史必须先 `git commit` 保存当前工作区**——禁止把未提交修改混入 rebase |
| **24** | **🚫 Merge 后必须验证按键面板**——`Panels/Keys.cs`/`KeyPanelViewModel.cs`/`KeyViewModel.cs`/`Views/Animation.cs` |
| **25** | **🚫 代码创建的 ContextMenu/Popup 必须设 PlacementTarget**——否则暗色模式无效 |
| **26** | **🚫 禁止移动 XAML 文件到子目录**——BAML 缓存冲突导致启动崩溃 |
| **27** | **🚫 移动资源文件后必须 grep 所有引用**——XAML 相对路径编译器不验证，运行时才炸 |
