### ✅ 第十二场（2026-06-18）— #10 FFT 频谱

| 项目 | 说明 |
|------|------|
| 频域 Model | `FreqModel`（`LinearAxis` 横轴 Bin/Hz + 纵轴 Magnitude），与 `TimeModel` 共享一个 `PlotView`，`plotView.Model` 换引用 |

| 窗函数 | 汉宁/矩形/汉明/布莱克曼 生效（`RecomputeFft` 乘入 buffer），与 CMSIS-DSP 对齐 |
| 采样率 | 侧面板输入 Hz → X 轴 Bin → Hz，频率范围/分辨率/基频全部 Hz 显示 |
| 频域 HUD | 右上角蓝色边框标签，显示数据源名 |
| 面板 | `rightPlotFreq`（频谱设置）+ `rightPlotDetailFreq`（频域指标：基频 Hz/幅度/THD/SNR/DC 偏置） |
| 视角切换 | `PerspectiveToggleStyle`（Ghost 描边），按钮 `⏱ 时域 → 频域 📶` / `📶 频域 → 时域 ⏱` |

- ⚠️ B3：`SetFftSource` 切 plot 源 → `_fftBuffer = new double[128]` 全零 → FFT 直线 → 只在尺寸变时重建
- ⚠️ B4：频域下 `UpdatePlotHud()` 仍在跑 → 加 `!_isFreqDomain` 守卫
