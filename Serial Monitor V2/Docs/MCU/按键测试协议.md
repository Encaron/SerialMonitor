# STM32 按键测试协议

> 对应 PC 端协议：`[key,name,state]`（state = `on` / `off`）
> 同时兼容纯文本格式：`on\r\n` / `off\r\n`

## 功能

接收 PC 端按键消息，执行 ON/OFF 动作并回复确认。

## C 代码

```c
/**
 * 按键测试处理 —— 支持协议包和纯文本两种格式
 *
 * 协议包格式: [key,按键名,状态]    例: [key,on,on]  [key,off,off]
 * 纯文本格式: on\r\n  /  off\r\n
 *
 * 回复格式: "按键测试处理正确,已执行：on/off，当前格式：文本/数据包"
 */
void Test_KeyHandle(void)
{
    if (!UART_GetRxFlag(Serial_PC))
        return;

    char *raw = UART_GetRxData(Serial_PC);
    if (!raw || raw[0] == '\0') return;

    const char *format;       // "文本" 或 "数据包"
    const char *action;       // "on" 或 "off"
    int execute_on = 0;       // 1=开, 0=关

    // ── 分支 1: 协议包格式 [key,xxx,xxx] ──
    if (raw[0] == '[')
    {
        char type[16];
        GetField(raw, 0, type, sizeof(type));

        if (strcmp(type, "key") != 0)
            return;  // 不是按键消息，忽略

        char name[32], state[16];
        GetField(raw, 1, name, sizeof(name));
        GetField(raw, 2, state, sizeof(state));

        if (strcmp(state, "on") == 0)
        {
            execute_on = 1;
            action = "on";
        }
        else if (strcmp(state, "off") == 0)
        {
            execute_on = 0;
            action = "off";
        }
        else
        {
            return;  // 未知状态，忽略
        }

        format = "数据包";
    }
    // ── 分支 2: 纯文本格式 on / off ──
    else
    {
        // 去掉末尾 \r \n
        char *p = raw + strlen(raw) - 1;
        while (p >= raw && (*p == '\r' || *p == '\n'))
            *p-- = '\0';

        if (strcmp(raw, "on") == 0)
        {
            execute_on = 1;
            action = "on";
        }
        else if (strcmp(raw, "off") == 0)
        {
            execute_on = 0;
            action = "off";
        }
        else
        {
            return;
        }

        format = "文本";
    }

    // ── 执行动作 (替换为你的硬件操作) ──
    if (execute_on)
    {
        // HAL_GPIO_WritePin(LED_GPIO_Port, LED_Pin, GPIO_PIN_SET);
    }
    else
    {
        // HAL_GPIO_WritePin(LED_GPIO_Port, LED_Pin, GPIO_PIN_RESET);
    }

    // ── 回复确认 ──
    Log("按键测试处理正确,已执行：%s，当前格式：%s\r\n", action, format);
}
```

## 依赖

- `UART_GetRxFlag(Serial_PC)` — 检查串口接收标志
- `UART_GetRxData(Serial_PC)` — 获取接收字符串
- `GetField(raw, index, buffer, size)` — 解析逗号分隔字段（复用滑杆测试已有函数）
- `Log(...)` — 串口 printf 宏
