## #8 调参工作台 — 踩坑记录

> 5 个文件，+323 / -20 行，~4 小时（含排查）

| # | 现象 | 根因 | 修法 |
|:--:|------|------|------|
| B1 | 打开抽屉瞬间 OxyPlot `NullReferenceException` 崩溃 | `InvalidatePlot(true)` 走 Background dispatch，与抽屉展开时 plotArea resize 触发的 OxyPlot 内部渲染同时跑，`GetClippingRect` 拿到空引用 | 三管齐下：① `InvalidatePlot(false)` 不强制同步渲染 ② `_backgroundRenderScheduled` 防重入锁 ③ `panelPlot.UpdateLayout()` 先完成布局再改 Visibility |
| B2 | Sliders 页滑杆拖不动 | `HandleSliderMessage` 每条 `[slider,...]` 调 `InitSliderPanel()` → 现在每次都跑 `RefreshSlidersUI()` 销毁重建，拖拽中控件被回收 | `InitSliderPanel(bool refreshUI=true)`，协议路径传 `false`，只确保 VM 存在不重建 UI |
| B3 | Sliders 页进去看不到滑杆 | `EnsureSliderVM()` 提前创建了 `_sliderVM` → `InitSliderPanel()` 的 `if (_sliderVM != null) return;` 把 UI 创建也跳过了 | 拆开 VM 创建和 UI 刷新：VM 只建一次，UI 每次 `refreshUI=true` 都跑 |
| B4 | 在 Plot 页开串口，调参按钮不出现 | `RefreshTuningDrawer()` 只在切标签页时调用，已在 Plot 页时没有触发点 | `[plot,...]` 和 `[slider,...]` 到达时，如果按钮还没显示且当前在 Plot 页，调一次 `RefreshTuningDrawer()` |
| B5 | 切出 Plot 页再切回，波形冻结不更新 | 非 Plot 期间数据照存但跳渲染（`IsActive=false`），切回来时 `_dirty=true` 没 flush，等下一次数据才渲 | 检测 `IsActive` false→true 跳变时调 `Flush()` |
| B6 | 调参按钮遮挡 X 轴时间标签 | 按钮在 `plotArea` 内部 bottom-center 浮层，和 OxyPlot 横轴标签重叠 | 移出 plotArea，单独一行（波形图和 GridSplitter 之间），140×24 PrimaryButtonStyle |

### 架构备忘

- **VM 共享**：调参抽屉的滑杆和 Sliders 页共用 `_sliderVM.Sliders[n]`，不是拷贝。步长/颜色/范围双向即时生效。
- **名字交集**：`_plotVM.GetChannelNames() ∩ _sliderVM.Sliders` → 只显示有波形曲线的滑杆
- **调度法**：`UseBackgroundRender=true` 时 `InvalidatePlot(false)` 走 `DispatcherPriority.Background`，让路给 MouseMove。拖拽流畅，波形帧率自然降到 15-25Hz，松手恢复满 30Hz
- **波形压扁不是 Bug**：抽屉弹出后绘图区物理高度减小，曲线变扁但数据不变。Y 轴 Zoom、CSV 导出、HUD 数值全部不受影响
- **`EnsureSliderVM()`**：轻量 VM 初始化（不建 UI），供调参抽屉在 Sliders 页未访问时使用。从 prefs.json 恢复已保存的滑杆配置

### 架构备忘 + 创意决策

- **双 Model 单视图**：`TimeModel` + `FreqModel`，`plotView.Model` 换引用。切换后各自保留缩放/滚动状态
- **PC 端自动 FFT**：`[plot,...]` 数据滑窗 buffer（128 点默认）→ 自研 radix-2 FFT（纯 C# ~50 行，零依赖）→ 实时频谱。用户 STM32 零改动就能看频域
- **`[fft,name,点数,bin0,...]` 命名协议**：给 FFT 加名字字段，兼容旧格式。数据源下拉框分 `📈 plot` 和 `📶 fft` 两类，选中命名 FFT 后不被自动 FFT 覆盖
- **数据源蓝色边框卡片**：`PrimaryBrush` 描边 + 15px 标签 + ↻ 刷新按钮（22×22），上下层布局防拥挤
- **采样率 → Hz**：侧面板加采样率输入框 → X 轴从 Bin 索引变 Hz，频率范围/分辨率/基频全部 Hz 显示
- **窗函数生效**：汉宁/矩形/汉明/布莱克曼四选一，`RecomputeFft` 内乘入 buffer。CMSIS-DSP 同名对齐
- **频域 HUD**：右上角 `PrimaryBrush` 边框标签，显示当前数据源名（`📈 Slider1` 或 `📶 ch1`），时域则走原有数值 HUD
- **PerspectiveToggleStyle**：Ghost 描边（`PrimaryBrush` 边框/文字 + 透明底），介于 tab 和按钮之间。按钮文案 `⏱ 时域 → 频域 📶` / `📶 频域 → 时域 ⏱`，箭头指向目标
- **暂停冻结两路**：`IsPaused` 同时冻结 `[plot,...]` 和 `[fft,...]`，切换视角只看不倒数据
