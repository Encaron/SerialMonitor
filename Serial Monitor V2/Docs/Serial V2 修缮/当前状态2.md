## 前情提要

项目始于江科大 WinForms 串口助手——用户改了三个默认值（115200 / 文本 / UTF-8），从此开启了"工具应该记住我怎么用"的基因。

**v1.0.0**（WPF 重写 UI，30+ 功能）→ **V2**（`[type,arg,...]` 协议系统 + OxyPlot 波形 + 4 控制面板 + .NET 8）。Phase 1-6 共 ~100 commits，踩坑无数（OxyPlot 不能写 XAML、ContextMenu 暗色失效、icon.ico 移动引发 7h 崩溃……），全部记录在 `../Serial V2 Initial/当前状态.md` 和 memory 中。

用户是嵌入式开发者（STM32/ESP32），不会 C#/WPF，只双击 exe。技术栈：WPF .NET 8 + OxyPlot 2.2 + AvalonEdit + 21 色双主题。编译 `dotnet publish -c Release`，产出 1.95MB 单文件 exe。已发布 GitHub v2.1.0。
