## 一、核心洞察

**不造 Canvas 渲染引擎。** 全屏 Plot 面板用 OxyPlot 已经跑了两年，滚动模式、Y 轴刻度、网格线、图例、HUD、缩放平移——全部现成的。

波形图卡片 = **全屏 Plot 面板缩小到 400×380 塞进传感卡**。

每张波形卡内部是一个 `PlotView`（C# 创建，遵守约束 #19），配自己的 `PlotModel` + `LineSeries`。多曲线 = 多 `LineSeries`，OxyPlot 原生支持。

| | Canvas 自绘路线 | **OxyPlot 嵌入（本方案）** |
|------|:--:|:--:|
| 滚动/扫描 | 手写 ~80 行 | 0（OxyPlot 内置 DateTimeAxis/LinearAxis） |
| 网格线 | 手画虚线 | 0（Axis 自带 MajorGridlineStyle） |
| Y 轴刻度 | 手写 TextBlock | 0（Axis 自带） |
| 多曲线 | 手画多 Polyline + 手写图例 | 0（`PlotModel.Series.Add` + Legend） |
| 缩放/平移 | ❌ | ✅ OxyPlot 鼠标交互 |
| 代码量 | ~550 行 | **~240 行** |
| 难度 | 中偏大 | **中** |

## 二、协议格式

### MCU → PC：波形数据上报

```
[sensor,waveform,卡片名,曲线名,值]
```

5 字段，比普通传感卡多一个"曲线名"：

| 位置 | 字段 | 示例 | 说明 |
|:--:|------|------|------|
| 1 | 大类 | `sensor` | 固定 |
| 2 | 子类型 | `waveform` | 新建 |
| 3 | 卡片名 | `示波器` | 一张卡可含多条曲线，卡名一致即归入同一卡 |
| 4 | 曲线名 | `P` | LineSeries 标题 = 图例显示名 |
| 5 | 值 | `0.5` | 数值，浮点 |

**多曲线示例：**

```
[sensor,waveform,示波器,P,0.5]
[sensor,waveform,示波器,I,0.3]
[sensor,waveform,示波器,D,0.1]
→ 三行数据，进同一张卡"示波器"，自动建三条曲线（绿/蓝/橙）
```

**空值处理：**

```
[sensor,waveform,示波器,P,,]  → P 曲线当前帧跳过（不追加数据点），线段不连
```

### 字段规则

- 卡名 = 协议第三字段 = 显示名。固件端命名，PC 端原样显示。
- 曲线名 = 协议第四字段。自动成为 LineSeries.Title。
- 值必发。空 → 该曲线该帧跳过，不画点。
- 无 aux 字段。中段留空用双逗号。
- 卡名唯一（`FindByName` 跨组搜），曲线名在单卡内唯一。

### 和全屏 Plot 的关系

```
[plot,P,0.5]                    → PlotViewModel.Series["P"]  → 全屏 OxyPlot 波形面板
[sensor,waveform,示波器,P,0.5]  → WaveformCard.Series["P"]   → 传感面板波形卡

两条数据流完全独立，和滑杆双轨（[slider,...] / [ctrl,slider,...]）同一模式。
MCU 端同时发两条即可。
```

## 三、卡片视觉

```
┌── 400×380 ───────────────────────────────────────────┐
│ ┌ 绿 ┬─────────────────────────────────────────────┐ │
│ │ 色  │ 示波器                    P: 0.5   I: 0.3 │ │  ← HUD：各曲线当前值
│ │ 竖  │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ │
│ │ 条  │  100 ┤  ── P        ╭─╮                   │ │  ← 图例叠在左上
│ │ 12  │      │  ── I     ╭─╯  ╰──                │ │     OxyPlot Legend
│ │     │   50 ┤  ── D   ╭╯                        │ │
│ │     │      │       ╭╯                           │ │  ← LineSeries 曲线
│ │     │    0 ┤╭─────╯                             │ │     无面积填充
│ │     │      └──────────────────────────────      │ │
│ │     │       ← 200 点 / 滚动 →                  │ │  ← 底部信息条
│ └────┴─────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────┘
```

| 属性 | 值 | 说明 |
|------|------|------|
| **尺寸** | 400×380 | 宽 = 2.5 个普通卡，高 ≈ 2 个普通卡 |
| **竖条色** | 绿 `#4CAF50` / 暗色 `#66BB6A` | 波形卡专属，不和 status 绿混淆 |
| **标题** | 左上角，13pt Bold，竖条色 | 卡片名 |
| **曲线色** | 色板轮转：绿→蓝→橙→紫→青→粉→循环 | 第一条绿 = 竖条色，后续自动分配 |
| **线宽** | 1.5px | 比全屏 Plot 的 2px 略细，适配缩小尺寸 |
| **图例** | OxyPlot Legend，卡内左上角 | 最多 6 条，超出缩略 |
| **HUD** | 右上角，曲线名: 当前值 | 每曲线一行，竖条色小号字 |
| **底部信息条** | 点数 / 滚动 / 采样率 | 8pt `TextMutedBrush` |
| **无面积填充** | 纯曲线 | — |
| **交互** | 鼠标缩放/平移（OxyPlot 原生） | 编辑模式下禁用 |

## 四、架构

```
数据流：
  串口 → ProtocolParser
    → RouteMessage: case "sensor" → RouteSensorMessage
      → SensorPanelViewModel.OnSensorMessage("waveform", cardName, curveName, value)
        → 找卡 FindByName(cardName)
        → 卡内找曲线 card.GetOrCreateSeries(curveName)
        → series.Points.Add(DateTimeAxis.ToDouble(now), value)
        → 裁剪 card.TrimPoints()

渲染：
  33ms DispatcherTimer → RefreshWaveformCards()
    → 遍历所有 waveform 卡
    → card.PlotModel.InvalidatePlot(true)
    → 更新 HUD 文字

侧栏详情：
  编辑模式 → 侧栏切详情面板（和滑杆卡同模式）
    → 曲线列表（名称/颜色/可见/删除）
    → 波形设置（模式/点数/Y轴）
    → 操作（清除/重置）
  
  正常模式 → 侧栏只读详情
    → 各曲线当前值 + min/max/avg
    → 📋 复制协议预览
```

### VM 层级

```
SensorPanelViewModel
  └── Groups[]
        └── SensorGroup
              └── Items[]
                    ├── SensorCardViewModel (Type="waveform")
                    │     ├── PlotModel (OxyPlot)         ← 每卡一个
                    │     ├── SeriesMap: Dict<string, LineSeries>
                    │     ├── MaxPoints: int = 200
                    │     ├── DisplayMode: Scroll
                    │     ├── YAutoRange: bool = true
                    │     ├── YMin / YMax: double
                    │     └── SampleRate: double
                    ├── "---" (换行标记)
                    └── SensorCardViewModel (其他类型)
```

### 数据路由（不动）

`RouteMessage` → `case "sensor"` 已扛所有子类型。waveform 走 `RouteSensorMessage` 的 else 分支 → `_sensorVM.OnSensorMessage("waveform", ...)`。协议路由零改动。

## 五、多曲线

### 自动建曲线

`OnSensorMessage("waveform", cardName, curveName, value)` 收到新数据时：

1. `FindByName(cardName)` → 无则建卡
2. `card.SeriesMap.TryGetValue(curveName)` → 无则建 LineSeries，自动分配颜色
3. 追加数据点
4. 裁剪旧点（MaxPoints）

### 曲线配色轮转

```
色板 = [#4CAF50, #2196F3, #FF9800, #9C27B0, #00BCD4, #E91E63]
       绿        蓝        橙        紫        青        粉

第一条曲线 = 竖条色（绿），后续按色板顺序分配。
相邻曲线保证不撞色。
```

### 共享 Y 轴

所有曲线共享同一个 LinearAxis。Y 自动范围 = 所有曲线数据的全局 min/max。取消自动后可手动设 YMin/YMax。

## 六、ViewModel 设计

### SensorCardViewModel 新增字段

```csharp
// ——— 仅 waveform 类型 ———

/// <summary>OxyPlot 模型（每卡一个）</summary>
public PlotModel PlotModel { get; set; }

/// <summary>曲线名 → LineSeries</summary>
public Dictionary<string, LineSeries> SeriesMap { get; } = new();

/// <summary>显示模式：Scroll / Sweep</summary>
public string DisplayMode { get; set; } = "scroll";

/// <summary>最大显示点数（50~1000）</summary>
public int MaxPoints { get; set; } = 200;

/// <summary>Y 轴自动范围</summary>
public bool YAutoRange { get; set; } = true;

/// <summary>Y 轴最小值（取消自动后生效）</summary>
public double YMin { get; set; } = 0;

/// <summary>Y 轴最大值（取消自动后生效）</summary>
public double YMax { get; set; } = 100;

/// <summary>采样率 Hz（仅显示用，不影响渲染）</summary>
public double SampleRate { get; set; }
```

### GetAccentHex 扩展

```csharp
"waveform" => isDark ? "#66BB6A" : "#4CAF50",  // 绿
```

### GetUnit 扩展

```csharp
"waveform" => "",  // 无单位，纯数字
```

### Update 扩展

waveform 类型走 default 分支（`Value = value; History.Enqueue(v)`），数据追加在 `RouteSensorMessage` 层完成（因为需要曲线名参数做二次路由）。

### 序列化扩展

```csharp
if (Type == "waveform")
{
    d["curves"] = SeriesMap.Keys.ToList();       // 曲线名列表（颜色由色板重新分配）
    d["mode"] = DisplayMode;
    d["maxPoints"] = MaxPoints;
    d["yAuto"] = YAutoRange;
    if (!YAutoRange)
    {
        d["yMin"] = YMin;
        d["yMax"] = YMax;
    }
    if (SampleRate > 0) d["sampleRate"] = SampleRate;
}
```

## 八、波形渲染引擎

### PlotView 创建

```csharp
// Panels/Sensors.cs — CreateWaveformCard(vm)

var plotModel = new PlotModel
{
    PlotAreaBorderThickness = new OxyThickness(0),
    PlotMargins = new OxyThickness(0),
    IsLegendVisible = true,
    LegendPosition = LegendPosition.LeftTop,
    LegendOrientation = LegendOrientation.Vertical,
    LegendFontSize = 9,
    TextColor = isDark ? OxyColors.White : OxyColors.Black,
    PlotAreaBackground = isDark ? OxyColor.Parse("#1E1E1E") : OxyColor.Parse("#FFFFFF"),
};

// X 轴 — 时间轴（和全屏 Plot 一致）
var xAxis = new DateTimeAxis
{
    Position = AxisPosition.Bottom,
    StringFormat = "HH:mm:ss",
    FontSize = 8,
    MajorGridlineStyle = LineStyle.Solid,
    MinorGridlineStyle = LineStyle.Dot,
    MajorGridlineColor = isDark ? OxyColor.Parse("#3A3A3A") : OxyColor.Parse("#E0E0E0"),
    TicklineColor = isDark ? OxyColor.Parse("#555555") : OxyColor.Parse("#CCCCCC"),
    TextColor = isDark ? OxyColor.Parse("#888888") : OxyColor.Parse("#666666"),
};

// Y 轴
var yAxis = new LinearAxis
{
    Position = AxisPosition.Left,
    FontSize = 8,
    MajorGridlineStyle = LineStyle.Dash,
    MinorGridlineStyle = LineStyle.Dot,
    // ... 同 X 轴色系
};

plotModel.Axes.Add(xAxis);
plotModel.Axes.Add(yAxis);

vm.PlotModel = plotModel;

var plotView = new PlotView
{
    Model = plotModel,
    Height = 340,  // 卡片 380 - 标题区 40
    // Width 由 Grid 列自动分配
};
```

### 数据追加

```csharp
// RouteSensorMessage 中：
if (vm.Type == "waveform")
{
    if (!vm.SeriesMap.TryGetValue(curveName, out var series))
    {
        var colorIdx = vm.SeriesMap.Count;
        var hex = WaveformPalette[colorIdx % WaveformPalette.Length];
        series = new LineSeries
        {
            Title = curveName,
            StrokeThickness = 1.5,
            Color = OxyColor.Parse(hex),
            MarkerType = MarkerType.None,
            LineStyle = LineStyle.Solid,
        };
        vm.PlotModel.Series.Add(series);
        vm.SeriesMap[curveName] = series;
    }

    series.Points.Add(new DataPoint(DateTimeAxis.ToDouble(DateTime.Now), v));

    // 裁剪旧点
    while (series.Points.Count > vm.MaxPoints)
        series.Points.RemoveAt(0);
}
```

### 刷新

```csharp
// 33ms DispatcherTimer 中，对 waveform 卡走 RefreshWaveformCards
private void RefreshWaveformCards()
{
    if (_sensorVM == null || !_sensorVM.IsActive) return;

    foreach (var group in _sensorVM.Groups)
    {
        foreach (var card in group.Cards.Where(c => c.Type == "waveform"))
        {
            if (card.PlotModel == null) continue;

            // Y 自动范围
            if (card.YAutoRange)
                RecalcYAxis(card);

            card.PlotModel.InvalidatePlot(true);

            // 更新 HUD 文字
            UpdateWaveformHud(card);
        }
    }
}
```

*注：`InvalidatePlot(true)` 走同步渲染。4 张波形卡同时刷新预计每帧 ~8-12ms（OxyPlot 软件渲染），33ms 帧预算够用。后续如果性能不够，可改为 dirty flag 节流。*

## 十一、风险与对策

| 风险 | 等级 | 对策 |
|------|:--:|------|
| 多 OxyPlot 实例 CPU 开销 | 低 | 3-4 张卡 = 4-5 个 PlotModel，OxyPlot 软件渲染每帧 ~2-3ms/个。33ms 预算够。超过 5 张可降为 dirty flag 节流 |
| PlotView 在 ScrollViewer 内鼠标事件冲突 | 低 | Bubble 事件模型——PlotView 的缩放平移事件在卡内消费，不冒泡到外层 ScrollViewer |
| 编辑模式下波形冻结 | 低 | 跟现有迷你波形同机制：`IsEditMode` → 数据照存不渲染 |
| 主题切换 PlotModel 颜色 | 低 | `RegisterThemePanel("Sensors", callback)` 遍历更新所有波形卡的 Axis/Legend/PlotArea 颜色 |
| 400px 宽卡在窄窗口下折行 | 中 | WrapPanel 折行后单独占一整行。组内可加换行标记 `---` 隔离波形卡和其他小卡。16:9 屏幕（1600px 宽）≈ 3 张波形卡并排，够用 |
