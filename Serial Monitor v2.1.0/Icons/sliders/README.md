# 滑杆轨道与滑钮素材

> **功能**：把 PNG 放进来 → 侧面板轨道/拇指下拉菜单各自独立出现你的风格。
> **轨道和拇指可以独立搭配**——用默认轨道 + 自定义拇指，或者反过来。

---

## 命名规则

| 文件 | 说明 |
|------|------|
| `track_风格名.png` | 轨道底板（横向长条）—— 可选，没有就用代码绘制 |
| `thumb_风格名.png` | 拇指/拖钮（正方形图标）—— 可选，没有就用代码绘制 |

**你可以只放一张**：比如只放 `thumb_mybtn.png` 不放 `track_mybtn.png`，那么轨道照常用代码画，只有拖钮换成你的图。反过来也行。

格式：**PNG**（推荐透明背景），文件名**小写英文 + 下划线**。

### 轨道尺寸

| 素材 | 推荐高度 | 推荐宽度 | 说明 |
|------|:---:|:---:|------|
| `track_xxx.png` | **4~12 px** | ≥ 200 px | 必须横向长条，不能正方形。高度决定轨道粗细；宽度随意，软件会拉伸铺满 |

```
正确（横向长条）：           错误（正方形）：
█████████████████████████    ┌────────┐
█████████████████████████    │ 210x210│  ← 横向拉伸后严重变形
   300×8                    └────────┘
```

### 拇指尺寸

| 素材 | 推荐尺寸 | 说明 |
|------|:---:|------|
| `thumb_xxx.png` | **16~48 px** 正方形 | 圆形/方形/任意图形都行；透明背景即可。决定拖钮的大小 |

---

## 示例

想加一个自定义拖钮 `mybtn`（只用默认轨道）：

```
Icons/sliders/
├── README.md
└── thumb_mybtn.png      ← 只有拇指，轨道用默认
```

侧面板：
```
轨道: [默认 ▾]           ← 代码画的 4px 灰条
拇指: [mybtn ▾]          ← 你的图片
```

---

## 内置两风格

| 风格 | 轨道 | 拇指 |
|------|:---:|:---:|
| 默认 | 4px 灰条，圆角 2 | 16×16 圆点，白色描边 2px |
| 极简 | 2px 灰条，圆角 1 | 12×12 圆点，白色描边 1px |

内置风格始终可用。放同名 PNG 文件可覆盖（`track_default.png` → 替代默认轨道的代码绘制）。

---

## 彩色填充条行为

填充条是**实心彩色矩形**，叠加在轨道图片上方。算法不变：宽度 = `(当前值-最小值)/(最大值-最小值) × 滑杆总宽`。

## 不想用了

删掉对应的 PNG → 重启软件 → 菜单里消失 → 之前选这个风格的滑杆自动回退到"默认"。
